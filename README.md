# Depression Risk Prediction Using Supervised Machine Learning

## Overview
This project uses three supervised machine learning models (Logistic Regression, Random Forest with SMOTE, and XGBoost) to predict whether a US adult respondent has ever been formally diagnosed with a depressive disorder.

## Repository Contents

| File | Description |
|------|-------------|
| `brfss2023_depression_MODELREADY.csv` | Cleaned, model-ready dataset derived from the 2023 CDC BRFSS public-use file. |
| `Data_Import_Clean.ipynb` | Notebook 1 - Raw data cleaning, missing value handling, and feature encoding. |
| `Main_Notebook.ipynb` | Notebook 2 - Model training, threshold tuning, cross-validation, and SHAP explainability. |

## How to Run
Run the notebooks in order:
1. `Data_Import_Clean.ipynb`
2. `Main_Notebook.ipynb`
